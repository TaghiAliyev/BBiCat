
 Run Application
 ===============

 Windows: 	Doubleclick the BicAT_v2.2.jar file 

 Linux:		type ./run_linux to start the application.

 Solaris:	type ./run_solaris to start the application.
 
	 
 possible problems:
 ==================

	 Make sure the environment variables are properly set!
 
	 CLASSPATH (should be ok)

	 LD_LIBRARY_PATH = ./lib/linux
			   ./lib/solaris
